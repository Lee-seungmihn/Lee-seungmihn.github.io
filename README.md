# Lee Seung-min's Personal Blog & Portfolio

이승민의 개인 블로그 및 포트폴리오 프로젝트입니다.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Design System:** Custom Style Guide based on `blog_settings.md`
- **Typography:** Pretendard, JetBrains Mono

## Project Structure
- `index.html`: 메인 랜딩 페이지 (포트폴리오)
- `pages/`: 서브 페이지 (수업자료, 스타일 가이드 등)
- `css/`: 기능별 CSS 파일 분리 (`reset`, `font`, `common`, `util`, `style`)
- `js/`: 기능별 JS 파일 분리 (`layout`, `service`, `event`)
- `blog_settings.md`: 프로젝트 작업 규칙 및 스타일 가이드 정의

## Development Rules
- 모든 컴포넌트는 `blog_settings.md`에 정의된 스타일 가이드를 준수합니다.
- CSS 변수를 활용하여 일관된 디자인 시스템을 유지합니다.
- 모바일 퍼스트 대응 및 웹 표준을 준수합니다.
