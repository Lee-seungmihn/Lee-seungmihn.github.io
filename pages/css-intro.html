CSS 기초 (Cascading Style Sheets)

CSS의 기본 개념과 문법, 선택자, 그리고 우선순위 규칙에 대한 학습 정리입니다.

1. CSS란 무엇인가?

CSS는 Cascading Style Sheets의 약자입니다.

  • HTML 태그들이 각종 미디어에서 어떻게 보일지를 정의할 때 사용합니다.
  • 스타일을 HTML 문서로부터 분리하는 것이 가능해집니다.

2. CSS를 사용하는 이유

HTML만으로 웹 페이지를 제작할 경우 다음과 같은 문제점이 발생합니다:

  • 세부 스타일을 일일이 따로 지정해야 해서 시간이 많이 소요됩니다.
  • 완성 후에도 스타일 변경 및 유지보수가 어렵습니다.

CSS의 장점:

  • W3C에서 만든 스타일 시트 언어 표준입니다.
  • 웹 페이지의 스타일을 별도의 파일로 저장하여 사이트 전체 스타일을 손쉽게 제어할 수 있습니다.
  • 웹 사이트의 스타일을 일관성 있게 유지하고, 유지보수가 쉬워집니다.

3. CSS 문법

선택자 {
    /* 선언부 */
    속성명: 속성값;
}

/* 예시 */
p {
    text-align: center;
    color: blue;
}

4. CSS 선택자

(1) 전체 선택자 (*)

스타일을 모든 태그에 적용할 때 사용합니다. Asterisk(*) 기호를 사용합니다.

(2) 태그 선택자 (태그명)

특정 태그가 쓰인 모든 요소에 스타일을 적용합니다.

(3) 클래스 선택자 (.클래스명)

특정 집단의 여러 요소를 한 번에 선택할 때 사용합니다. 같은 class=""를 가지는 요소들을 모두 선택합니다.

(4) 아이디 선택자 (#아이디명)

특정 요소를 선택할 때 사용합니다. 하나의 문서에 중복된 이름의 아이디가 있어서는 안 됩니다.

(5) 그룹 선택자 (쉼표 구분)

여러 선택자에 같은 스타일을 적용할 경우 쉼표(,)로 구분해서 나열합니다.

5. 캐스케이딩 (Cascading)

하나의 요소는 하나 이상의 CSS 선언에 영향을 받을 수 있으며, 충돌을 피하기 위해 우선순위가 필요합니다.

(1) 중요도 (Source Order)

  1. 인라인 스타일: HTML 태그 안에 style 속성
  2. 내부 스타일 시트: HTML 문서 안에 style 태그
  3. 외부 스타일 시트: 별도 CSS 파일
  4. 웹 브라우저 기본 스타일

(2) 명시도 (Specificity)

선택자를 구체적으로 사용할수록 우선순위가 높아집니다.

  1. !important
  2. 아이디 선택자 (#id)
  3. 클래스 선택자 (.class)
  4. 태그 선택자 (tag)
  5. 전체 선택자 (*)
  6. 상속받은 속성 (Inherited)

(3) 선언 순서

나중에 선언된 스타일이 우선 적용됩니다.
